# Plan: Add a simple greeting to the CLI home screen

## Goal

Display a static welcome greeting (e.g. "Welcome to Kilo!") on the TUI home screen, positioned between the ASCII logo and the prompt input.

## Current Layout (home.tsx)

The home screen renders vertically in this order:

1. Spacer (flexGrow)
2. Spacer (height 4)
3. **Kilo ASCII logo** (`<Logo />`)
4. Spacer (height 1)
5. **Prompt input** (max-width 75)
6. KiloNews + Tips
7. Spacer (flexGrow)
8. Toast
9. Footer bar (directory, MCP, version)

## Changes

### 1. Create a new `Greeting` component

**File:** `packages/opencode/src/kilocode/components/greeting.tsx` (new file)

This is Kilo-specific, so it goes in the `kilocode` directory (no `kilocode_change` markers needed).

The component will:

- Render a centered `<text>` element with "Welcome to Kilo!" in the theme's muted text color
- Be a simple, stateless SolidJS component using `useTheme()` for styling

```tsx
import { useTheme } from "@tui/context/theme"

export function Greeting() {
  const { theme } = useTheme()
  return <text fg={theme.textMuted}>Welcome to Kilo!</text>
}
```

### 2. Add the Greeting to the Home screen

**File:** `packages/opencode/src/cli/cmd/tui/routes/home.tsx` (existing, shared file — needs `kilocode_change` markers)

Insert the `<Greeting />` component between the logo and the prompt. Specifically, replace the current spacer `<box height={1}>` between logo and prompt with:

```tsx
{
  /* kilocode_change start */
}
;<Greeting />
{
  /* kilocode_change end */
}
```

This places the greeting directly below the logo and above the prompt, centered by the parent `alignItems="center"`.

Add the import at the top of the file:

```tsx
import { Greeting } from "@/kilocode/components/greeting" // kilocode_change
```
