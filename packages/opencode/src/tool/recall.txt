Search and read past conversations from any project on this machine. Use this to recall previous work, find how something was implemented before, or retrieve context from other projects.

Two modes:
1. **Search** - Find sessions by title keyword across all projects. Returns a list of matching sessions with their project, title, and last updated time. Use this first to locate relevant conversations.
2. **Read** - Retrieve the full transcript of a specific session by ID. Returns the conversation messages (user prompts and assistant responses) so you can understand what was discussed and done.

Usage notes:
  - Search matches against session titles using case-insensitive substring matching
  - Results include sessions from ALL projects on this machine, not just the current one
  - Reading a session from a different project requires permission
  - Use search mode first to find session IDs, then read mode to get the full conversation
  - Session transcripts can be large; prefer searching first to narrow down which session to read
